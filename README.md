# P8P Symfony Bundle

> **⚠️ Development Status**
>
> This project is under active development and is not yet ready for production use.
> APIs and features may change without notice.

The Bundle provides Symfony integration for the P8P Kubernetes client.
It handles configuration, dependency injection, and provides a registry for managing multiple
Kubernetes clients with support for DSN-based configuration.
